# House of Hands Web App

### WebApp used to control Smart Home devices

#### Environment setup:
1. Install the API following :
  `nasm qemu gcc make`

2. Build the Toolchain (on x64BareBones project directory):
  `cd Toolchain`
  `make all`

3. Build the Kernel (on x64BareBones project directory):
  `make all`

4. Run the kernel
  `./run.sh`
